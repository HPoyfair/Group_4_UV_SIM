## Overview

UVSim is a simple virtual machine written in C#.  
It simulates a BasicML computer with 100-word memory and an accumulator-based CPU.

The simulator loads a BasicML program from a text file and the instructions loaded from memory.

## Requirements

- .NET 8.x.x.x SDK installed
